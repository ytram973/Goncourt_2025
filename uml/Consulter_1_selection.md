# Consulter la 1ère sélection

## Acteur principal
Utilisateur

## Précondition
La base est initialisée avec la 1ère sélection (table `g_selection` + `g_book_selection`).

## Scénario nominal
1. L'utilisateur lance l'application
2. L'application affiche le menu principal
3. L'utilisateur choisit l'option 1 – Afficher la 1ère sélection
4. Le système interroge la base
5. Le système affiche la liste des livres de la première sélection avec :
    - titre
    - prénom + nom de l'auteur
    - nom de l'éditeur
